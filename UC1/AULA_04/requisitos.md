# SISTEMA DE: RH
## PROBLEMAS
A empresa tem que controla números de funcionarios controle de ferias e jornada


## OBJETIVO
Tem que criar um sistema para organizar o nome de usuário, número de pontos e número expecifico de entrada e saida


## USUÁRIOS
são os navegadores e os funcionarios e estagiarios



## REGRAS DE NEGÓCIO
O limite de atraso ate 10 minutos
funcionario não pode tirar muitos dias de ferias seguidas senão pode descontar 50 centavos de salario
funcionarios tem direito ao horario de almoço de 11:30 até 12:00
o limite de ferias são de 15 dias 
## REQUISITOS FUNCIONAIS
sistema deve permitir cadastar os funcionarios
sistema deve registra entrada e saida dos funcionarios
sistema deve registra agendamentos
sistema deve registra a ausensia  dos funcionarios
sistema deve permitir consultar os agendamentos das ferias
sistema deve descontar os atrasados sobre o salario



## REQUISITOS NÃO FUNCIONAIS
sistema deve ter interface simples
sistema deve funcionar em computadores e celulares
sistema deve proteger os dados do usuario
sistema deve carregar rapidamente
sistema deve ter layout responsivo
